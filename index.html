Welcome

Hello and welcome to my personal website. I have created this as a method of providing more informtion about me and what I have done. I have included a section devoted to my previous works, a section detailed about me, and several links at the bottom to my email and other pages.

Projects

Below is a list of my personal projects, one of which is this website. This is an expanding list as more projects are started and worked on. Not all are in a finished, polished state. Some are ongoing or abandoned projects.

Gifinity

A team project for my Intro to Object-Oriented Design in Java class. I worked with three other members on an application that could take segments of video and convert them into animated GIFs. This project is not available on Github and is currently private. If you want more information, feel free to contact me by e-mail.


Hexadecimal Clock

This project was started around September of 2016. A project started out of curiosity, the main focus was to simply see 24 hour time in hexadecimal instead of decimal. I consider it noteworthy since it is the first project where I worked with timers and time formatting. In addition, this project has great potential for expandability. For example, one plan is to develop a mobile application that makes use of it.

This project is ongoing and on hold.


Bradfordy-Eight

Working version of the game 2048 in Java. This game was created while learning Java in high school. This started out as a way to test my skills in Java when I first started programming. I added some basic interface afterwards. This project was my first time trying to use swing elements.


Phoneword Checker

A project in C++ that takes a 7-digit phone number and tests if it can find a matching phoneword for it. This is a project that I am currently attempting to rework into an object-oriented design.


Personal Website

About

My name is Bradley Sawyer Johnson. I was born and raised around Montgomery, Alabama. I first started using computers when I was young with small educational games. I became more familiar with using computers through technology classes and clubs during elementary and middle school. These gave me skills using common work applications such as Word, PowerPoint, and Excel in my classes and basic programming through clubs. In high school, I took my first computer science course introducing me to the occupation. After learning HTML, Java, and core computer science concepts, I was convinced of my future in computer science.

Currently, I am an undergraduate student at the University of Alabama in Huntsville seeking a bachelor's degree in computer science and a bachelor's degree in mathematics. One of my biggest interests right now is machine learning and its capabilities especially when applied to cybersecurity. Computer science has given me an understanding and fondness for problem-solving. It is always incredibly satisfying when a difficult, sanity-draining issue is resolved. Currently, I am searching for opportunities to apply the knowledge and skills I've gained into larger-scale projects while expanding my knowledge and skills.
